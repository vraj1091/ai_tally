## 🚨 TROUBLESHOOTING GUIDE

### If Tally Won't Connect:
1. Check if Tally is running
2. Verify company is open
3. Confirm Gateway is enabled (F1)
4. Check port 9000
5. Try localhost reset button

### If Data Shows ₹0:
1. Check if company has data
2. Refresh data manually
3. Verify ledger types in Tally
4. Check date ranges

### If Charts Don't Load:
1. Refresh the page
2. Clear browser cache
3. Check browser console
4. Try different browser

### If App is Slow:
1. Check internet connection
2. Close unnecessary tabs
3. Restart backend server
4. Clear cache
