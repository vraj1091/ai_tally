### 2. AI Assistant Features

- **Natural Language Queries**: "What were my top 5 customers last month?"
- **Intelligent Insights**: Automatic detection of trends and anomalies
- **Predictive Analytics**: Revenue forecasting and cash flow predictions
- **Smart Recommendations**: AI-driven business suggestions
- **Document Analysis**: Upload and analyze financial documents

### 3. Cloud Bridge Technology

Our proprietary **TallyConnector** technology enables:
- Secure WebSocket connection between local Tally and cloud
- No data stored on cloud - real-time streaming only
- Works behind firewalls without port forwarding
- Supports data volumes up to 2GB per request
- 99.9% uptime reliability

### 4. Data Sources

| Source | Description |
|--------|-------------|
| **Live Tally** | Direct connection when on same network |
| **Bridge Mode** | Cloud access via TallyConnector |
| **Backup Files** | Upload .tbk, .xml backup files |

## Technology Stack

### Backend
- **Framework**: Python FastAPI
- **Database**: PostgreSQL / SQLite
- **AI Model**: Ollama with Phi4:14b LLM
- **Vector Store**: ChromaDB for RAG
- **Real-time**: WebSocket with 2GB message support

### Frontend
- **Framework**: React 18 with Vite
- **UI**: Tailwind CSS with modern components
- **Charts**: Recharts for data visualization
- **State**: React Context + Hooks

### Infrastructure
- **Cloud**: AWS EC2 / Any cloud provider
- **Proxy**: Nginx reverse proxy
- **Container**: Docker + Docker Compose
- **Security**: JWT authentication, HTTPS
