TROUBLESHOOTING
================================================================================

IF YOU STILL GET 500 ERRORS:

1. CHECK YOUR LOGIN
   - Make sure you're logged in (test2@mail.com)
   - Token might have expired - logout and login again

2. REFRESH BROWSER COMPLETELY
   - Press: Ctrl + Shift + Delete
   - Clear "Cached images and files"
   - Click "Clear data"
   - Then: Ctrl + F5 to reload

3. CHECK SERVER STATUS
   - Backend: http://localhost:8000/docs (should show API docs)
   - Frontend: http://localhost:5173 (should load app)

4. CHECK FILE FORMAT
   - File must be .xml (or .tbk, .001, .zip)
   - Max size: 100 MB
   - Must be valid Tally XML export

IF XML PARSING FAILS:

Error Message: "No Tally data found in XML file"
Solution: Ensure XML contains company/ledger data
   - In Tally: Gateway → Import/Export → Export
   - Select your company
   - Choose "XML" format
   - Select "All Masters & Transactions"
   - Export to file

Error Message: "Invalid XML format"
Solution: File might be corrupted
   - Try exporting again from Tally
   - Check file opens in text editor without errors

Error Message: "File too large"
Solution: Reduce file size
   - Export specific date range
   - Export only required masters
   - Or zip the XML file
