## 🚀 **How to Use**

### **Access Dashboard Hub**
```
http://localhost:5173/dashboards
```

### **Switch Dashboards**
1. Click on any dashboard in the sidebar
2. Use category filters at the top
3. Search for specific dashboards

### **Change Chart Types**
1. Click "Change Chart Type" button on any chart
2. Select category (Bar, Line, Area, etc.)
3. Choose specific chart type
4. Chart updates instantly!

### **Time Range Selection**
- Click Day/Week/Month/Quarter/Year buttons
- Data automatically filters
- Charts update in real-time
